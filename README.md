## 实验基本功能要求
用MVC模式实现以下功能
- **注册及登录：** login and register page
- **主页留言列表：** home page、message list page                       
- **发布留言：** post message page
- **留言详情：** message details page                                                  
- **发表评论：** post comment for a message                                   
- **搜索留言：** search message page
